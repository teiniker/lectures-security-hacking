# Base64 Encoding and Decoding 

## Using the Bash

```
$ echo  'Security&Hacking' | base64
U2VjdXJpdHkmSGFja2luZwo=
```

```
$ echo 'U2VjdXJpdHkmSGFja2luZwo=' | base64 --decode
Security&Hacking
```
